### Icon图标引入方案

##### 1.svg

使用[icons8](https://icons8.cn/)生成整张svg图像 ，按生成页提示名称引入svg

优点：可以任意改变颜色，缩放不会模糊,经编译后无bug

<i class="icon-2">
  <svg>
  	<use xlink:href=".../path/to/Default.min.svg#icon-2"></use>
  </svg>
</i>

<style>
  /* 改变大小和颜色 */
  .icon-2 {
    color: red；
    font-size: 24px；
  }
</style>
